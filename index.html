About Our Project

This projects presents an advisory system for users, arranging sets of daily meals. One of the main advantages of the system is generating divers menus satisfying personalized nutrition standards. An algorithm is applied to the task of generating menus.
A system based on Machine Learning algorithm that takes information from the user and predicts the probability of risk for particular health condition such as Depression, Diabetes, Thyroid, Cardiac problems.
